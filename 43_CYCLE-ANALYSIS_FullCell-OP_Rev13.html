📌 분석 파라미터 정의

  • Full-cell Capacity Retention (80%): 초기 방전용량(3rd cycle 기준) 대비 80% 방전용량 지점의 cycle 수
  • Symmetric-cell OP: 각 cycle의 최대전압 - 최소전압 차이(mV)를 계산하여 기준값(기본 1000mV) 도달 cycle 표시
  • OP 그래프: 모든 cycle의 OP를 충전(양수)/방전(음수)로 표시하여 0 기준 위아래 대칭 형태
  • 지원 장비: Neware (Ah→mAh 자동변환), Won-A Tech (mAh), Biologic
  • 자동 분석: Full-cell과 Sym-cell 데이터를 자동으로 구분하여 함께 분석
